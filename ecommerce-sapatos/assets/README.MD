# SyntaxWear - Tênis e Sneakers Online

Bem-vindo à SyntaxWear, o seu destino online para os tênis e sneakers mais estilosos e confortáveis. Este projeto é uma loja virtual desenvolvida para proporcionar uma experiência de compra intuitiva e agradável, com um design moderno e responsivo.

## 🚀 Visão Geral

A SyntaxWear é uma loja de e-commerce fictícia, criada como um projeto de desenvolvimento web para demonstrar habilidades em HTML, CSS e design responsivo. O site apresenta uma variedade de calçados, organizados em categorias e exibidos em um layout de grade dinâmico.

## ✨ Recursos

- **Design Responsivo:** A loja se adapta perfeitamente a diferentes tamanhos de tela, desde dispositivos móveis até desktops.
- **Layout Moderno:** Interface limpa e atraente, focada na experiência do usuário.
- **Navegação Intuitiva:** Menus e links claros para facilitar a busca por produtos.
- **Grade de Produtos Dinâmica:** Exibição de produtos em uma grade que se ajusta ao layout da página.
- **Categorias de Produtos:** Seções dedicadas para diferentes estilos de tênis, como casual, esportivo, moderno e futurista.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Para a estrutura semântica do site.
- **CSS3:** Para estilização e layout, utilizando o modelo de caixas flexíveis (Flexbox) e grade (Grid).
- **Design Responsivo:** Media queries para garantir a adaptabilidade em diferentes dispositivos.
